# Correction langues EN/AR

Cette version remplace les pages `/en/` et `/ar/` par deux pages écrites séparément, sans logique de traduction automatique ni changement de langue JavaScript.

Fichiers prioritaires à remplacer sur GitHub :
- `en/index.html`
- `ar/index.html`
- `index.html`
- `service-worker.js`
- `sitemap.xml`

Les pages sont volontairement indépendantes pour éviter les mélanges FR/EN/AR et les problèmes d'affichage mobile RTL.

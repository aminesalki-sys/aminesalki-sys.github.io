Déploiement GitHub Pages

1. Remplacez les fichiers de votre dépôt par le contenu de ce dossier.
2. Gardez le dossier assets/ et le dossier articles/ à la racine.
3. Les images du portfolio sont appelées avec ces noms exacts :
   - assets/site aviation solidaire.png
   - assets/Site mariage Afrah Darna.png
   - assets/site sono-ned medical.png
   - assets/site soprofeu.png
   - assets/site switch care.png
   - assets/site taxi navette.png
4. La page blog est blog.html et chaque article est dans articles/.
5. sitemap.xml et robots.txt sont inclus pour le SEO.
